认知心理学家把人的知识看作三个层层嵌套的圆形区域：

  • 最内层【舒适区】：已经熟练掌握的知识、技能；

  • 最外层【恐慌区】：暂时无法学会的知识、技能；

  • 中间层【学习区】：适合进行高针对性练习，达成学习目的的区域。

不同学生的知识结构、基础各不相同，因此“中间层”也不尽相同，但高考的“筛选”标准不会照顾个人！

每个学生应该准确的定位自己的“学习区”，并努力的拓展。该专栏总结的内容，是每个高分学生必经的“学习区”。

数学学科特点

数学的三大学科特点 + 数学具有逻辑的严谨性 + 数学具有概括性，抽象性 + 数学具有连续性，系统性

数学学习的4大原则 + 主动性和积极性原则 + 循序渐进原则 + 及时反馈原则 + 独立思考和创造性原则

结合认知规律和学科特点，提几个学习要求： + 对自己有个正确的认识，梳理知识的盲区； + 遵循学科特点，循序渐进，打好基础； + 认真务实，不可眼高手低，吃透、理解透彻； + 在“学习区”勤加练习，向外拓展；（从似懂非懂开始）

“题海”无边

“题海”的目的： + 构建知识体系：辅助课堂内容理解、掌握、熟练； + 更新、补充、更正知识体系：查缺补漏，检验学习成果，找到知识掌握的不足； + 积累解题经验、应试技巧，提高成绩；

“题海”简单粗暴，相对低效；浩瀚但不精准，缺乏总结。

乘风破浪

书山题海、考试中，有一些频频出现的典型问题，应该是重点掌握。

该课程总结归纳了高中数学中的一些典型问题，同时针对每个典型问题，总结了不同的思路和方法。目的是让高中生能更精准的学习，将这些极具“代表性”的问题逐一击破。

数学知识具有结构性、系统性，知识点是相互联系的，需要借助严谨的思维、方法建立联系；知识点间建立的联系越多，理解越准确，掌握也越扎实。

典型问题的结论，可以直接使用之外，其方法往往也可用于解决其他问题，都是需要积累的。
